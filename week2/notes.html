template literals

  • allows both types of quotes to be used in string without having to escape
  • variables can be inserted directly into using ${}

Symbol

  • const varName = Symbol();
  • primitive datatype but allows non conflict naming

Infinity

  • special error value that represents any number to bit for the JS to deal with

Type Coercion

  • happens when the operands of an operator are of different types. In this case, JavaScript will attempt to convert one operand to an equivalent value of the other operand’s type.

Logical operators

  • can use double negation (!!) to check if value is truthy or falsy

Bitwise operators

  • any number with a bitwise operator is converted into a 32 bit int.
  • The bitwise NOT operator ~ will convert the number to a 32-bit integer, then change all the 1 s to 0 and all the 0 s to 1 s
  • The bitwise AND operator, & , will convert both numbers into binary, and returns a number that in binary has a 1 in each position for which the corresponding bits of both operands are 1 s
  • There is also the bitwise OR operator, | , which will convert both numbers into binary and return a number that in binary has a 1 in each position for which the corresponding bits of either operands are 1 s
  • Another operation is the bitwise XOR operator, ^ , which stands for “eXclusive OR”. This will convert both numbers into binary and return a number that in binary has a 1 in each position for which the corresponding bits of either operands are 1s, but not both 1 s
  • The bitwise shift operators, << and >>, will move the binary representation a given number of places to the right or left, which effectively multiplies ordivides the number by powers of two

CHAPTER 4

  • forEach() look through an array and invoke a callback on each argument.
  • map() also loops through each item and invokes a callback function on each item and returns an array with the result
  • reduce() combines each result to return a single value. usually takes two parameters. the first one represents the accumulated value and second the current value.
  • filter() returns an array that contains items from array called on, if the callback returned true on the item
